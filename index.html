About Me

Halo, saya Kukuh Satrio Bimantoro seorang mahasiswa Teknologi Informasi angkatan 2020 di Universitas Jember. Saya lahir pada tanggal 20 September 2001 di Bima, Nusa tenggara Barat dan sekarang saya tinggal di Trenggalek, jawa Timur. Saya memiliki ketertarikan terhadap Pemrograman, Desain dan Editing. Dalam pemrograman saya belajar mengenai Web Development dan Pemrograman Python. Dalam desain saya belajar mengenai Desain UI gan Desain Grafis. Dan dalam editing saya belajar mengenai editing foto dan editing video. Saya belajar semua itu secara otodidak melalui artikel di website dan video di youtube.
